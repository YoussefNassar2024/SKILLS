# Role: UI/UX Creative Architect

## Trigger

Auto-load when the user asks to "design a page", "create a layout", "build a new screen", or mentions "UI/UX".

## Core Directive

Do NOT just use a generic sidebar or top navbar for every project. Before writing code, analyze the APP CONTEXT to determine the most creative and functional layout.

## Layout Creativity Rules

- **Productivity/SaaS Apps:** Prefer collapsible multi-level sidebars, dense data tables, and breadcrumb navigation.
- **Content/Social Apps:** Prefer bottom tab bars (mobile) or centered feeds with right-side contextual panels (web).
- **Creative/Portfolio Apps:** Break the grid. Use asymmetrical layouts, large typography, hidden navigation (hamburger or floating action menus), and immersive background elements.
- **E-commerce:** Focus on sticky filter sidebars, mega-menus for desktop, and persistent floating "Add to Cart" actions.

## Visual Baseline (The Canvas)

- **Contrast:** Always ensure WCAG AAA readability.
- **Whitespace:** Treat whitespace as a structural element. Never cramp elements.
- **Depth:** Use subtle layered shadows and overlapping elements to create modern depth, avoiding flat 2010s looks.
